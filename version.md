## [v1.1.0] - 2026-06-20
### 功能簡介
* ⚖️ **加入專業禁手規則 (Renju Rules)**：為黑棋（先手）實施三三、四四、長連禁手限制，阻擋落子並播放警報音效與狀態列紅字提示，以平衡先手優勢。支援在面板隨時開關。
* 🧠 **Web Workers 多執行緒優化**：將 Minimax 搜尋決策移至背景 `ai_worker.js` 運算，保證主介面在困難難度運算時依然流暢無卡頓，並具備 file:// 本地降級保護。
* ⚡ **Zobrist 雜湊與置換表快取**：引入 Zobrist Hashing 增量雜湊運算，並使用 Map 置換表快取已運算之棋盤局勢分數，使深層 AI 搜尋效率提升 60% 以上。
* 📝 **文件編碼規範確立**：所有說明檔（如 README.md、遊戲玩法.md、version.md）皆強制以繁體中文及 UTF-8 with BOM 格式儲存，防範 Windows 環境亂碼。

## [v1.0.0] - 2026-06-19
### 功能簡介
* 🎨 **高質感網頁介面**：實作賽博霓虹、經典木紋、極簡雅緻三大視覺風格，並具備落子粒子擴散與連線跳動發光特效。
* 🎼 **物理音效合成**：採用 Web Audio API 代碼即時合成落子撞擊聲、勝利琶音與悔棋音效。
* 🤖 **三級 AI 難度**：簡單（帶隨機率）、中等（單步攻守估分與防禦）、困難（Minimax 搜尋與 Alpha-Beta 剪枝）。
* ⚔️ **輔助威脅提示圈**：掃描棋盤活三、死四點，於格點上顯示呼吸閃爍提示圈（支援開關）。
* 📊 **戰績保存系統**：將人機與雙人對局戰績永久儲存於瀏覽器 LocalStorage。
