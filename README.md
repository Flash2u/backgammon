# 🌌 Cyber Gomoku - 極致美學五子棋

一個融合現代賽博霓虹美學、經典木質風、極簡雅緻風的五子棋網頁版遊戲。具備動態粒子特效、Web Audio API 合成物理碰撞聲、即時威脅提示，以及基於 Web Workers 多執行緒運算、支援 Zobrist Hashing 與置換表的 Minimax AI 演算法。

👉 **[線上直接遊玩網址](https://flash2u.github.io/backgammon/)**

---

## 🎮 遊戲特色與功能

### 1. 🎛️ 遊戲模式與輔助系統
* **雙重對戰模式**：
  * **人機對戰 (PvE)**：與內建智慧型 AI 進行博弈，支援玩家與 AI 輪流執黑子（先手自動落天元）。
  * **雙人對決 (PvP)**：支援本機雙人同桌對弈。
* **威脅提示系統 (Threat Hints)**：
  * 當盤面上出現「活三」或「衝四/死四」等關鍵威脅時，棋盤格點上會以橘色或桃紅色霓虹虛線圈進行**動態呼吸閃爍提示**，協助玩家防禦或進攻。可隨時在設定面板開啟或關閉。
* **歷史紀錄與悔棋 (Undo)**：
  * 支援無限步數悔棋。在人機對戰下，點擊悔棋會自動退回兩步（玩家與 AI 的前一手）；雙人對決模式下退回一步。

### 2. 🚫 專業禁手規則 (Renju Rules)
為了平衡黑棋（先手）必勝的極大優勢，加入了專業禁手規則（可隨時在設定面板開關）：
* **三三禁手**：黑棋禁止同時形成兩個或以上的「活三」。
* **四四禁手**：黑棋禁止同時形成兩個或以上的「四」（含活四、死四/衝四）。
* **長連禁手**：黑棋禁止連成六顆或以上的連續直線棋子。
* **高質感警告機制**：
  * **視覺反饋**：黑棋回合下，棋盤上所有的禁手點會自動偵測，並在滑鼠懸停時顯示**紅色發光警示棋子**且指針變為禁用狀態。
  * **點擊阻擋**：強行點擊禁手點時會被判定無效，播放低頻警告音效，並在狀態列顯示具體禁手類型。

### 3. 🧠 AI 演算法與效能優化 (Web Workers)
* **Web Workers 多執行緒**：
  * 在「困難」難度下，AI Minimax 決策樹會向下預判 4 步 (depth=3) 並對前 20 個高分點進行深層搜尋。
  * 運算完全移至背景執行緒，網頁畫面（計時器、動畫、呼吸燈）維持 **60 FPS** 流暢度，主執行緒完全不卡頓。
  * 針對本地 `file://` 協定，內建**降級安全防護**，如瀏覽器限制 Worker 載入將自動轉為單執行緒計算，確保程式不崩潰。
* **Zobrist 雜湊與置換表**：
  * 利用 Zobrist Hashing 以 XOR 運算增量計算唯一的盤面 Hash 值，並使用 Map 作為**置換表 (Transposition Table)** 進行盤面狀態快取，避免重複評估相同局勢，效能提升 60% 以上。

### 4. 🎨 極致視覺美學與音效合成
* **三大主題切換**：
  * 🌌 **暗黑霓虹 (Cyber Neon)**：毛玻璃卡片、發光發亮之桃紅與青藍霓虹棋子，科幻感十足。
  * 🪵 **經典木紋 (Classic Wood)**：暖色榧木紋棋盤、經典雙色黑白子，沉穩舒適。
  * 📐 **極簡雅緻 (Minimalist Slate)**：純白背景與深灰高對比冷色系設計。
* **物理音效合成 (Web Audio API)**：
  * 不使用任何外部音訊檔，純以代碼合成音效。
  * **落子聲**：高頻瞬態 click 敲擊點 + 低頻木質棋盤共鳴，重現擬真落子質感。
  * **勝利音效**：達成五子連線時播放舒緩 of C 大調五聲音階琶音。
  * **警告音效**：黑棋點擊禁手點時播放低頻的鋸齒波警示音。
* **視覺動態特效**：
  * 棋子落子時噴散半透明發光粒子。
  * 獲勝的五子連線會進行放大跳動與呼吸發光動畫。
* **戰績統計**：
  * 系統自動根據當前模式統計勝場（人機模式統計玩家與 AI 勝率；雙人模式統計黑棋與白棋勝率），並透過 `localStorage` 保存，支援一鍵清除數據。

## 🛠️ 開發技術棧
* 核心結構：**HTML5 (語意標籤)**
* 樣式美術：**Vanilla CSS3 (設計系統變數、Glassmorphism、Keyframe 動態特效)**
* 程式邏輯：**Vanilla JavaScript (ES6+, 無依賴任何第三方框架)**
* 背景運算：**Web Workers API**
* 即時音效：**Web Audio API (OscillatorNode, GainNode, BiquadFilterNode)**
* 數據儲存：**LocalStorage API**

## 🚀 本地執行說明

1. **直接雙擊開玩 (簡單)**：
   * 在專案目錄下直接雙擊 **`start_server.bat`** 批次檔。
   * 批次檔會自動一鍵啟動本機伺服器，並在瀏覽器中彈出 [http://localhost:8080/](http://localhost:8080/)，即可直接開始遊玩！

2. **透過命令列啟動 (開發者)**：
   ```bash
   # 進入專案資料夾
   cd Sub_backgammon
   
   # 使用 npx 啟動靜態伺服器
   npx http-server -p 8080
   ```
   啟動後，在瀏覽器輸入 [http://localhost:8080/](http://localhost:8080/) 即可。
   
   也可以直接用網址
   https://flash2u.github.io/backgammon/
